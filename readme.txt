== Description ==
 
 
* 2023年6月16日に施行された改正電気通信事業法における3号事業者が外部送信情報を効率的に作成するためのプラグイン.
* 利用中のサービスにチェックを入れてショートコードを挿入するだけで対応完了です。
 
== Installation ==

1. プラグイン -> 新規追加から「電気通信事業法.jp」を検索・インストールして有効化します
2. 利用規約に同意しメールアドレスを登録します
3. 利用中のサービスにチェックを入れます
4. ショートコード[電気通信事業法]を、表示を挿入したい箇所に記述します
5. 表示が適切に行われているか確認します
 
 
== Frequently asked questions ==
 
 
Q. LiteSpeed Cacheが入っている環境でサービスを選択できない
LiteSpeed Cacheが導入されている環境で利用中のサービスを選択できない現象を確認しております。

これはREST APIの応答がキャッシュされていて、選択中のサービス変更前の応答をすぐに返す設定になっていることが原因です。

対処法は

WordPressに管理者のアカウントでログインします。
左側管理メニューの「LiteSpeed Cache」 の中にある「キャッシュ」ページを開きます。
「[1]キャッシュ」タブの「REST APIキャッシュ」をオフに変更して、変更を保存します。
左側管理メニューの「LiteSpeed Cache」の中にある「ツールボックス」ページを開きます。
「[1]パージ」タブの「すべてをパージする – LSCache」をクリックしてキャッシュを削除します。
これで選択したサービスが表に表示されるようになっています。
